# Framer 日本語化 Chrome拡張

Framerのインターフェースを日本語化するChrome拡張機能です。

> **注意:** これは非公式の個人開発プロジェクトです。Framer社とは一切関係ありません。ご利用は自己責任でお願いします。

## インストール方法

### 1. ダウンロード

このリポジトリをダウンロードします。

- 「Code」ボタン → 「Download ZIP」をクリック
- または `git clone` でクローン

```bash
git clone https://github.com/hgryk/framer-ja-extension.git
```

### 2. ZIPを解凍

ダウンロードしたZIPファイルを解凍してください。

### 3. Chromeに拡張機能を読み込む

1. Chromeで `chrome://extensions` を開く
2. 右上の「デベロッパーモード」をONにする
3. 「パッケージ化されていない拡張機能を読み込む」をクリック
4. 解凍したフォルダを選択

### 4. 完了

[Framer](https://framer.com) を開くと、インターフェースが日本語化されます。

## 更新方法

1. 最新版をダウンロード（または `git pull`）
2. `chrome://extensions` を開く
3. この拡張機能の「更新」ボタンをクリック
